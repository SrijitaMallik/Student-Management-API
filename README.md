# Student Management System  
A complete **Student Management Application** built using:  
- **ASP.NET Core Web API** (Backend)  
- **Angular 18** (Frontend)

This project allows users to perform CRUD operations on student data using a simple and clean UI. The backend exposes REST APIs, and the frontend consumes these APIs to display and manage student information.
